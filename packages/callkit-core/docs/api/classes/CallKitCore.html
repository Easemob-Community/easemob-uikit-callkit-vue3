    Class CallKitCore

    CallKitCore 框架无关的通话信令核心。

    职责：

    1. 管理单聊/群聊通话生命周期
    2. 通过事件回调通知上层所有决策点
    3. 不直接操作任何 RTC SDK，RTC 由上层通过适配器或事件自行处理
