# 铃声文件说明

⚠️ **铃声文件需要自行准备**

请将以下文件放置在此目录：

1. `outgoing_ringtone.mp3` - 呼出铃声（拨打电话时播放）
2. `incoming_ringtone.mp3` - 来电铃声（收到通话邀请时播放）

## 推荐铃声来源

- [freesound.org](https://freesound.org/) - 免费音效库
- [zapsplat.com](https://www.zapsplat.com/) - 免费音效
- 自己录制或设计

## 格式要求

- 格式：MP3
- 时长：建议 2-5 秒
- 音质：建议 128kbps 或更高
- 大小：建议不超过 100KB

## 使用示例

```javascript
// 在 Vue 项目中使用
const outgoingRingtone = new Audio('/sounds/outgoing_ringtone.mp3');
const incomingRingtone = new Audio('/sounds/incoming_ringtone.mp3');

// 播放呼出铃声
outgoingRingtone.loop = true;
outgoingRingtone.play();

// 停止铃声
outgoingRingtone.pause();
outgoingRingtone.currentTime = 0;
```
